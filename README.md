# 🌍 Climate Change Analysis

This project analyzes temperature and rainfall trends over the past 50 years.

## 📊 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset
Climate dataset containing yearly temperature and rainfall data.

## 🔍 Key Analysis
- Data cleaning and preprocessing
- Handling missing values
- Trend visualization
- Rolling average smoothing

## 📈 Results
The analysis shows long-term trends in climate patterns.
